# 问道紫微 - 紫微斗数分析助手

## 应用简介

问道紫微 是一款专业的紫微斗数排盘应用，致力于还原紫微斗数排盘的文字版本，为用户提供纯粹、准确的命理工具体验。我们专注于传统紫微斗数文化的数字化呈现，方便用户随时随地查看和分析命盘。

## 主要功能

### 1. 命盘解析
- 自动生成专业紫微斗数命盘
- 基于用户出生信息进行精准展示
