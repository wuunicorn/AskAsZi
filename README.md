# AskZiweiiPhone - 紫微斗数分析助手

## 应用简介

AskZiweiiPhone 是一款专业的紫微斗数分析应用，基于先进的 AI 技术，为用户提供准确的命理解读和分析服务。我们致力于将传统的紫微斗数知识与现代科技相结合，为用户带来便捷、专业的命理分析体验。

## 主要功能

### 1. 智能命盘分析
- 自动生成专业紫微斗数命盘
- 基于用户出生信息进行精准分析
- 提供详细的命盘解读说明

### 2. AI 智能问答
- 智能解答紫微相关问题
- 个性化的命理咨询服务
- 准确理解用户意图，提供专业解答

### 3. 运势预测
- 提供精准的运势分析
- 多维度的运程预测
- 针对性的人生规划建议
